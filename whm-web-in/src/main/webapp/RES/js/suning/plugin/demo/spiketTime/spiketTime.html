使用说明:

  1. 使用时引入sunign.spiketTime.js
  2. 初始化： J_spiketTime(id, startTime, endTime); 参数说明：id: 时间DIV id; startTime: 距离开始时间的毫秒数，endTime:距离结束时间的毫秒数
    如：J_spiketTime('Sn_spike_time',(0*60*60+25*60+0)*1000,(1*60*60+25*60+0)*1000); 距离开始时间0小时25分0钞，结束时间1小时25分0钞。
Note: 如已进入倒记时，则 startTime 可设置为0，或负数均可。
