使用说明:

  1. 使用时引入:suning.WdatePicker.js即可
  2. Validator.checkEmail($('#email').val()) 功能: Email校验, 参数: EMAIL内容, 返回: true/false;
  3. Validator.checkIdcard($('#idcard').val()) 功能: 身份证号码校验, 参数: 身份证号码, 返回: true/false;
  4. Validator.checkMobile($('#mobile').val()) 功能: 手机号码校验, 参数: 手机号码, 返回: true/false;
  5. Validator.checkPostCord($('#post').val()) 功能: 邮政编码校验, 参数: 邮政编码内容, 返回: true/false;
  6. Validator.checkDate($('#date').val()) 功能: 日期格式校验, 参数: 日期, 返回: true/false;
